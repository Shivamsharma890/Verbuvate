# Verbuvate – Language Translator

Verbuvate is a smart, modern language translator designed to help you explore,
understand, and learn languages with ease and accuracy and the Verbuvate makes 
language translation simple and fun — explore different languages, learn new 
words, and discover the world through language.

This is a React + Vite project that uses RapidAPI to translate text.
Features:
- Convert text between languages
- Supports multiple language options
- Fast and simple UI

Technologies:
- React.js
- Vite
- RapidAPI Translation API
